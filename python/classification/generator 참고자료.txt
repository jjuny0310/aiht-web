# 푸쉬업 데이터 생성
좌우반전을 하건 안하건 비디오의 화면에 나오는
방향이 left면 left푸쉬업 데이터로 생성
right면 right푸쉬업 데이터로 생성
ㄴ비디오영상의 방향을 잘보고 푸쉬업전용 좌우반전을 켯다 껏다 해야함
(하나의 비디오로 right, left 두개의 모델데이터를 생성해야 하니까)
